## 1. 美的方向性

### トーン & ムード

| 項目 | 方向性 |
|------|--------|
| トーン | Professional-Clean（プロフェッショナル＆クリーン） |
| ムード | 信頼感・効率性・知的さ |
| 参照イメージ | Notion + Ahrefs + Linear のハイブリッド |
| 差別化ポイント | データの美しい可視化 × 直感的なコンテンツ制作フロー |

### デザイン哲学

- **データファースト**: 数字とグラフが主役。装飾は最小限
- **フロー重視**: 分析→提案→生成の流れが画面遷移で自然に体感できる
- **2ブログ切り替え**: カラーテーマで現在どちらのブログを操作しているか直感的に判別
- **プログレス可視化**: AI処理中のステップを視覚的にフィードバック

### タイポグラフィ

| 要素 | フォント | Weight | サイズ |
|------|---------|--------|--------|
| 見出し (H1-H4) | Noto Sans JP | 600, 700 | 2rem〜1.125rem |

## 3. UIコンポーネント方針

### ライブラリ

| 項目 | 選定 |
|------|------|
| コンポーネント | shadcn/ui |
| スタイリング | Tailwind CSS v4 |
| アニメーション | motion/react |
| アイコン | Lucide React |
| チャート | Recharts |
| リッチエディタ | Tiptap |

### コンポーネント設計方針

- shadcn/ui をベースに、最小限のカスタマイズ
- `cn()` ユーティリティで条件付きクラス結合
- チャート・グラフは Recharts でデータ可視化
- 記事編集は Tiptap（WYSIWYG）でリッチな編集体験
- AI生成中のストリーミング表示対応

### 主要カスタムコンポーネント

| コンポーネント | 用途 |
|---------------|------|
| `BlogSwitcher` | マーケブログ/美容ブログ切り替え（アクセントカラー変化） |
| `ArticleOutlineEditor` | H1/H2/H3のドラッグ&ドロップ構成エディタ |
| `SEOScoreCard` | SEOスコアをゲージ表示 |
| `CompetitorComparisonTable` | 競合記事の並列比較テーブル |
| `PlatformPreview` | Note/X展開時のプレビュー（各媒体のUI再現） |
| `RankTracker` | 順位推移の折れ線グラフ |
| `CannibalizationMap` | カニバリ検出の関連図 |
| `CTAConfigurator` | CTAブロックの設定UI |

## 4. レスポンシブ方針

### ブレークポイント

| サイズ | 幅 | 対応 |
|--------|-----|------|
| Desktop (Primary) | 1280px+ | フル機能 |
| Laptop | 1024px〜1279px | サイドバー折りたたみ |
| Tablet | 768px〜1023px | 簡易ダッシュボード |
| Mobile | 〜767px | 閲覧のみ（記事生成は非推奨） |

### 方針

- **デスクトップファースト**（SEO分析・記事作成はデスクトップがメイン）
- タブレットではダッシュボード確認と記事の簡易編集に対応
- モバイルは順位トラッキングの確認に限定

```
+--------+----------------------+-------------------+
```

- 3カラムレイアウト: サイドバー + エディタ + SEOプレビュー
- SEOプレビューはリアルタイム更新

## 6. アニメーション & インタラクション

### 方針

| 項目 | ルール |
|------|--------|
| トランジション | 200ms以内 |
| プロパティ | transform, opacity のみ（compositor props） |
| AI生成フィードバック | ストリーミングテキスト + パルスアニメーション |
| チャート | 初回表示時にフェードイン + スケールアップ |
| ページ遷移 | 即時（アニメーションなし） |

### 禁止事項

- レイアウトプロパティのアニメーション（width, height, margin, padding）
- 過剰なバウンス・スプリングエフェクト
- ブラーエフェクト（backdrop-blur は 8px 以下のみ許可）

## 7. ダークモード

- OS設定に連動 + 手動切り替え
- 全コンポーネントでダークモード対応必須
- チャートの配色もダークモード用パレットを用意
- コントラスト比 4.5:1 以上を維持

## 9. 禁止パターン（Anti-Patterns）

| カテゴリ | 禁止 | 理由 |
|---------|------|------|
| フォント | Inter, Roboto, Arial | AI slop、日本語非対応 |
| カラー | 紫グラデーション on 白背景 | AI slop |
