## Custom Components

### BlogSwitcher

ヘッダー内配置。マーケブログ/美容ブログの切り替え。

```typescript
interface BlogSwitcherProps {
  blogs: Blog[];
  currentBlog: Blog;
  onSwitch: (blog: Blog) => void;
}
```

- カラードット + ブログ名表示
- 切り替え時にCSS変数 `--accent` を更新
- Zustand でグローバルstate管理

### ArticleOutlineEditor

記事構成のドラッグ&ドロップエディタ。

```typescript
interface OutlineNode {
  id: string;
  level: 'h1' | 'h2' | 'h3';
  text: string;
  summary: string; // 書くべき内容の要約
  children?: OutlineNode[];
}

interface ArticleOutlineEditorProps {
  outline: OutlineNode[];
  onChange: (outline: OutlineNode[]) => void;
  onApprove: () => void;
}
```

- H2/H3のドラッグ&ドロップ並び替え
- インライン編集（クリックで編集モード）
- H3追加/削除ボタン
- 各ノードに「書くべき内容」のトグル表示

### SEOScoreCard

SEOスコアのゲージ表示。

```typescript
interface SEOScoreCardProps {
  score: number; // 0-100
  details: {
    keywordDensity: number;
    wordCount: number;
    headingStructure: boolean;
    metaDescription: boolean;
    internalLinks: number;
    faqSchema: boolean;
  };
}
```

- 円形ゲージ（Recharts PieChart）
- スコアに応じた色変化（90+=緑, 70-89=琥珀, <70=赤）
- 詳細項目のチェックリスト

### CompetitorComparisonTable

競合記事の並列比較テーブル。

```typescript
interface CompetitorArticle {
  rank: number;
  url: string;
  title: string;
  wordCount: number;
  headings: HeadingNode[];
  coOccurrenceWords: string[];
  hasFaqSchema: boolean;
}

interface CompetitorComparisonTableProps {
  articles: CompetitorArticle[];
  targetKeyword: string;
}
```

- 横スクロール対応
- 見出し構成の視覚的比較
- 共起語のハイライト表示

### PlatformPreview

Note/X横展開時の各媒体プレビュー。

```typescript
interface PlatformPreviewProps {
  platform: 'note' | 'x_long' | 'x_post';
  content: string;
  cannibalizationScore: number;
}
```

- Note: Note風のカード表示
- X長文: X風の投稿カード表示
- Xポスト: 3-5投稿のカードリスト
- カニバリスコアバッジ表示

### RankTracker

順位推移の折れ線グラフ。

```typescript
interface RankDataPoint {
  date: string;
  rank: number;
  clicks: number;
  impressions: number;
}

interface RankTrackerProps {
  data: RankDataPoint[];
  keyword: string;
}
```

- Recharts LineChart
- Y軸反転（1位が上）
- ホバーでツールチップ（順位+クリック数+表示回数）
- 期間フィルタ（7日/30日/90日）

### CannibalizationMap

カニバリ検出の関連図。

```typescript
interface CannibalizationNode {
  articleId: string;
  title: string;
  keyword: string;
  platform: 'blog' | 'note' | 'x';
}

interface CannibalizationEdge {
  source: string;
  target: string;
  score: number; // 0-100
}
```

- ネットワークグラフ表示
- エッジの太さ=カニバリスコア
- ノードの色=プラットフォーム別
- クリックで詳細パネル表示

### CTAConfigurator

CTAブロックの設定UI。

```typescript
interface CTAConfig {
  type: 'line' | 'mail_magazine' | 'consultation' | 'download';
  position: 'middle' | 'end' | 'both';
  title: string;
  description: string;
  url: string;
  isActive: boolean;
}
```

- カードUIで設定一覧表示
- ドラッグ&ドロップで位置変更
- プレビュー表示
