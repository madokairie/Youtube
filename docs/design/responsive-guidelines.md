# SEO Article Factory - Responsive Guidelines

**作成日**: 2026-03-06

---

## Strategy: Desktop First

SEO分析・記事作成はデスクトップがメインの作業環境。
デスクトップで最高の体験を提供し、タブレット・モバイルは段階的に機能を制限。

---

## Breakpoints

| 名前 | 幅 | 対応レベル |
|------|-----|-----------|
| Desktop XL | 1536px+ | Full features |
| Desktop | 1280px〜1535px | Full features |
| Laptop | 1024px〜1279px | Sidebar collapsed |
| Tablet | 768px〜1023px | Simplified dashboard |
| Mobile | 〜767px | View-only (tracking) |

---

## Layout Adaptation

### Desktop (1280px+)

```
+--------+----------------------------+----------------+
| Sidebar| Main Content               | Side Panel     |
| 240px  | fluid                      | 320px (optional)|
+--------+----------------------------+----------------+
```

- フルサイドバー（240px、アイコン+テキスト）
- 記事作成: 3カラム（サイドバー + エディタ + SEOプレビュー）
- ダッシュボード: KPIカード4列

### Laptop (1024px〜1279px)

```
+----+--------------------------------------+
| 64 | Main Content                         |
| px | (wider)                              |
+----+--------------------------------------+
```

- サイドバー自動折りたたみ（64px、アイコンのみ）
- 記事作成: 2カラム（エディタ + SEOプレビュー折りたたみ可能）
- ダッシュボード: KPIカード3列

### Tablet (768px〜1023px)

```
+--------------------------------------+
| Header + Hamburger Menu              |
+--------------------------------------+
| Main Content                         |
+--------------------------------------+
```

- サイドバー → ハンバーガーメニュー（Sheet）
- 記事作成: 1カラム（エディタのみ、SEOプレビューはタブ切り替え）
- ダッシュボード: KPIカード2列
- テーブル: 横スクロール

### Mobile (〜767px)

```
+--------------------------------------+
| Header + Hamburger                   |
+--------------------------------------+
| Simplified Content                   |
+--------------------------------------+
| Bottom Navigation (3 items)          |
+--------------------------------------+
```

- 閲覧のみ（記事生成は非推奨、警告バナー表示）
- ダッシュボード: KPIカード1列
- 順位トラッキングの確認に限定
- ボトムナビ: ダッシュボード / トラッキング / 設定

## Component Responsive Behavior

| コンポーネント | Desktop | Tablet | Mobile |
|---------------|---------|--------|--------|
| BlogSwitcher | ヘッダー内ドロップダウン | 同左 | 同左 |
| KPICards | 4列グリッド | 2列 | 1列スタック |
| DataTable | フル表示 | 横スクロール | カードリスト化 |
| ArticleOutlineEditor | ドラッグ&ドロップ | 同左 | 表示のみ |
| TiptapEditor | 3カラム表示 | 1カラム | 非表示（警告） |
| SEOPreviewPanel | サイドパネル常時表示 | タブ切り替え | 非表示 |
| RankTracker | フルチャート | 簡易チャート | スパークライン |
| CompetitorTable | フル表示 | 横スクロール | アコーディオン |
| PlatformPreview | 3カラムプレビュー | タブ切り替え | スタック |

## Touch Targets

- 全インタラクティブ要素: 最小 44px x 44px
- ボタン間のマージン: 最小 8px
- テーブル行のタップ領域: 行全体
