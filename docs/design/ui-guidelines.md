**Tone**: Professional-Clean

## Typography Guidelines

### Principles

- **Noto Sans JP** で日本語を美しく表示
- データ・数値は **JetBrains Mono** で視認性確保
- 見出しは **600-700 weight** でメリハリ
- 本文は **0.875rem (14px)** でデータ密度とのバランス

### Application

| 要素 | フォント | サイズ | Weight |
|------|---------|--------|--------|
| ページタイトル | Noto Sans JP | 2rem (32px) | 700 |
| セクション見出し | Noto Sans JP | 1.5rem (24px) | 600 |
| カード見出し | Noto Sans JP | 1.125rem (18px) | 600 |
| 本文 | Noto Sans JP | 0.875rem (14px) | 400 |
| KPI数値 | JetBrains Mono | 2rem (32px) | 500 |
| 順位・スコア | JetBrains Mono | 0.875rem (14px) | 500 |
| ラベル・キャプション | Noto Sans JP | 0.75rem (12px) | 500 |

## Color Guidelines

### Principles

- Slate系のニュートラルカラーをベースに、ブログテーマカラーでアクセント
- データ可視化: 緑=ポジティブ、琥珀=注意、赤=ネガティブ の3色コード
- ダークモードではコントラスト比 4.5:1 以上を維持

### Semantic Colors

| 意味 | Light Mode | Dark Mode |
|------|-----------|-----------|
| 順位上昇 | Emerald `#059669` | `#34d399` |
| 順位横ばい | Slate `#64748b` | `#94a3b8` |
| 順位下降 | Red `#dc2626` | `#f87171` |
| リライト推奨 | Amber `#d97706` | `#fbbf24` |
| 新記事提案 | Blue `#2563eb` | `#60a5fa` |

## Motion Guidelines

### Principles

- 200ms以下のトランジション
- `transform` と `opacity` のみ（compositor props）
- AI生成中はパルスアニメーションでフィードバック
- ページ遷移アニメーションは**なし**（生産性ツールは即時遷移）

### Patterns

| 場面 | アニメーション | Duration |
|------|---------------|----------|
| ボタンhover | opacity 0.8 | 100ms |
| カード展開 | scale(1) + opacity | 200ms |
| モーダル表示 | translateY(8px) + opacity | 200ms |
| AI生成中 | pulse (opacity 0.5-1.0) | 1500ms loop |
| ストリーミングテキスト | typewriter effect | per character |
| チャート初回表示 | scale(0.95→1) + opacity | 300ms |
| トースト通知 | translateX(100%→0) | 200ms |

## Layout Patterns

### Sidebar Navigation

```
[Logo]
─────────────
[BlogSwitcher]
─────────────
Dashboard
Analysis
Competitors
Suggestions
Articles
Distribute
Tracking
─────────────
Settings
```

- アイコン + テキストの2列構成
- 折りたたみ時はアイコンのみ（64px幅）
- アクティブ状態はブログテーマカラーのバックグラウンド

### Data Cards

```
┌──────────────────────┐
│ Label          [icon] │
│                       │
│  1,234          +12%  │
│  (JetBrains)  (green) │
│                       │
│ ▁▂▃▄▅▆▇ (sparkline)  │
└──────────────────────┘
```

- 左上: ラベル、右上: アクションアイコン
- 中央: 大きな数値（mono）+ 変化率（semantic color）
- 下部: スパークライン（オプション）

### Table Design

- shadcn/ui DataTable ベース
- 行hover: `bg-muted/50`
- ソート可能カラムにアイコン表示
- ページネーション: 20件/ページ
- 順位カラムは色分け表示（1-3位=金、4-10位=緑、11-30位=琥珀、31位以下=赤）

## Component Interaction Patterns

### AI Generation Flow

```
[Generate Button] → [Loading State: パルス + 進捗テキスト]
                  → [Streaming: テキスト逐次表示]
                  → [Complete: 編集可能状態]
```

- 生成中はボタン無効化 + キャンセルボタン表示
- 進捗を段階テキストで表示（「競合分析中...」→「構成生成中...」→「完了」）
- ストリーミング中はスクロール自動追従

### Blog Switcher

- ヘッダー内にドロップダウン
- 切り替え時、CSS変数（--accent）を変更
- 200ms のカラートランジション
- 現在のブログ名 + カラードット表示

## Anti-Patterns (NEVER)

- Inter, Roboto, Arial（日本語アプリなのでNoto Sans JP）
- 紫グラデーション on 白背景
- ヒーローセクション（ツールアプリなので不要）
- ページ遷移アニメーション
- 過剰なドロップシャドウ
- グロー効果
- レイアウトプロパティのアニメーション
